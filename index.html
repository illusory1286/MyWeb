  1. CSS選擇是 : 使用Class選擇器去控管不同標籤的顏色與文字大小。
  2. margin-top:10px;margin-bottom: 10px;/* 控制div區塊上下各留10px */
  3. padding: 10px;/* 控制內部文字與邊框間隔 */
  4. 表格第一列使用虛擬選擇器
  5. div.className : 只限制div標籤
  6. 使用html 5的語法,對以後SEO(搜尋)優化會很好的幫助
